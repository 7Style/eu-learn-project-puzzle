# JavaScript Learning Game

## Overview
This JavaScript-based game is designed to teach students programming concepts in an interactive and engaging way. It features multiple levels, each introducing different challenges and programming principles.

## Features
- Interactive game board with dynamic elements.
- Multiple levels with increasing complexity.
- Custom commands for character control.
- Obstacles and gameplay mechanics that evolve with each level.

## Setup
To start the game, clone the repository and open `app.html` in a web browser.

```bash
git clone [Repository URL]
cd [Project Directory]
npm start 

and go to http://localhost:3000 in your browser

```

## How to Play
- **Level 1:** Basic movements and command execution.
- **Level 2:** Introduction of time limits and more complex paths.
- **Level 3:** Custom path creation and obstacle placement.
- **Level 4:** Two-player mode, where one player creates the path and obstacles, and the other tries to solve it.

## Game Structure
- `app.html`: The main HTML file that sets up the game's layout.
- `styles.css`: CSS file for styling the game's appearance.
- JavaScript Files:
  - `app.js`: Initializes the game and manages level selection.
  - `config.js`: Contains configuration settings for the game.
  - `commands.js`, `game.js`, `konfetti.js`: Handle game logic, commands, and effects.
  - `level2.js`, `level3.js`, `level4.js`: Define the logic for different levels of the game.
  - `store.js`: Manages data storage for the game.

## Customization
Feel free to modify the code to add new features, change game mechanics, or update the styling. This project is a great platform for experimenting with JavaScript and game development.
